Butterworth low-\/pass filter, fourth order.

Starting from analogue prototype transfer functions, using Tustin (bilinear) transform to arrive at digital approximation.

Butterworth filters are \textquotesingle{}maximally flat\textquotesingle{} before cut-\/off frequency, i.\+e. in passband.

Gain of any order Butterworth low-\/pass filter at cut-\/off frequency is -\/3dB (1/sqrt(2)). \hypertarget{butterworth__lpf_8h_autotoc_md9}{}\doxysubsubsection{\texorpdfstring{However, attenuation slope is n \texorpdfstring{$\ast$}{*} (-\/20d\+B/dec), where n is the filter order.}{However, attenuation slope is n * (-20dB/dec), where n is the filter order.}}\label{butterworth__lpf_8h_autotoc_md9}
